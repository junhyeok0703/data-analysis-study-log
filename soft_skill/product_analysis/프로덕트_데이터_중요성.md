## 프로덕트 데이터의 중요성

1. **사용자 경험 최적화**
   - 자주 쓰는 기능은 접근성 강화, 사용 빈도 낮은 기능은 개선·제거
   - *Melon* : 자주 듣는 플레이리스트를 메인에 배치
   - *Kakao Talk Labs* : 토글로 기능 별도 배포 → 선호 확인 후 정식 적용
2. **성과 객관화**
   - 메인 배너의 클릭률·전환율을 **AB Test**로 검증 → 효율 비교
3. **비용 절감·업무 효율**
   - 불필요한 풀배포 방지 (예: 번역 기능 30 %만 테스트)
   - 실패 기능 제거 시 분석·수정 비용 절감
