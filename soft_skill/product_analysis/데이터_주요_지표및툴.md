## 데이터 조회 & 주요 툴

| 지표          | 정의                           |
| ------------- | ------------------------------ |
| **DAU**       | Daily Active Users (중복 제거) |
| **PV**        | Page Views (중복 허용)         |
| **수익/비용** | 매출·광고·서버비 등            |

### 대표 툴 비교

| 툴            | 특화 포인트                                     |
| ------------- | ----------------------------------------------- |
| **GA4**       | 웹+앱 통합, 마케팅 퍼포먼스 추적, 퍼널 시각화   |
| **Amplitude** | 세션 리플레이, 퍼널·코호트 심화, 과거 행동 변화 |
| **Mixpanel**  | 이벤트 기반 분석, AB Test, 실시간 코호트        |
| **Vuzable**   | UX 히트맵, 스크롤·클릭·마우스 움직임 영상 재생  |

> 목적·요구사항에 맞춰 단일 또는 조합 사용 권장.
